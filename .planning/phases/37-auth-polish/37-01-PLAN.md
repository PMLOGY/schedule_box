Convert the auth layout from a server component to a client component with glass treatment. The current layout is 24 lines using plain `Card` + `bg-muted/40`. Replace it with:

1. Add `'use client'` directive at the top (required for Motion). This is safe because the child page.tsx files (login, register, forgot-password, reset-password) are server components that export `metadata` -- Next.js renders page.tsx server-side and passes content as `children` to the client layout.

2. Replace imports: Remove `Card, CardContent` from `@/components/ui/card`. Add:
   - `import { motion } from 'motion/react';`
   - `import { GradientMesh } from '@/components/glass/gradient-mesh';`
   - `import { GlassPanel } from '@/components/glass/glass-panel';`
   - Keep `LocaleSwitcher` and `ThemeToggle` imports.

3. Replace the outer div `bg-muted/40` with just `min-h-screen flex items-center justify-center relative`. Add `<GradientMesh preset="auth" />` as the first child (it renders as fixed, full-screen, behind content via `-z-10`).

4. Keep the top-right controls div but add `z-10` to ensure it's above the gradient mesh.

5. Wrap the card content area in a `motion.div` with entrance animation:
   - `initial={{ opacity: 0, y: 20 }}`
   - `animate={{ opacity: 1, y: 0 }}`
   - `transition={{ duration: 0.4, ease: [0.22, 1, 0.36, 1] }}`
   The motion.div wraps BOTH the branding header (ScheduleBox h1 + subtitle) and the GlassPanel, so the entire centered block animates in together.

6. Replace `<Card className="shadow-lg"><CardContent className="pt-6">{children}</CardContent></Card>` with `<GlassPanel intensity="heavy" className="p-6 rounded-xl">{children}</GlassPanel>`.

7. Form inputs inside child components (login-form, register-form, etc.) are NOT modified -- they already use the standard Input component with `bg-background` which keeps them opaque per AUTH-02.

```
div (min-h-screen flex center relative)

  div (absolute top-4 right-4 z-10)
    LocaleSwitcher + ThemeToggle
  div (w-full max-w-md px-4)
    motion.div (fade+slide entrance)
      div (text-center mb-8 branding)
      GlassPanel intensity="heavy" (p-6 rounded-xl)
        {children}
```
