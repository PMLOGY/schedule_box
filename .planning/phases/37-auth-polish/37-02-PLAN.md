**2. Create NEW file `apps/web/components/shared/glass-shimmer.tsx`:**
A reusable glass shimmer skeleton building block. Structure:
```typescript
import { cn } from '@/lib/utils';

interface GlassShimmerProps extends React.HTMLAttributes<HTMLDivElement> {
  className?: string;
}

export function GlassShimmer({ className, ...props }: GlassShimmerProps) {
  return (
    <div
      className={cn(
        'rounded-xl glass-surface-subtle overflow-hidden relative',
        className,
      )}
      {...props}
    >
      <div
        className="absolute inset-0 animate-shimmer rounded-xl"
        style={{
          backgroundImage:
            'linear-gradient(90deg, transparent 0%, rgba(255,255,255,0.06) 50%, transparent 100%)',
          backgroundSize: '200% 100%',
        }}
      />
    </div>
  );
}
```
The shimmer wave is a CSS gradient animation sliding horizontally over a glass-surface-subtle base. The `rgba(255,255,255,0.06)` provides a subtle light sweep visible on both light and dark backgrounds.

**3. dashboard-grid.tsx -- Add Motion stagger animation to KPI stat cards:**
Add `import { motion } from 'motion/react';` and import `GlassShimmer` from `@/components/shared/glass-shimmer`.

Define variant objects OUTSIDE the component (module scope, not re-created on render):
```typescript
const containerVariants = {
  hidden: {},
  show: {
    transition: {
      staggerChildren: 0.05, // 50ms between each card
    },
  },
};

const cardVariants = {
  hidden: { opacity: 0, y: 12 },
  show: {
    opacity: 1,
    y: 0,
    transition: {
      duration: 0.3,
      ease: [0.22, 1, 0.36, 1], // Custom ease-out curve
    },
  },
};
```

Replace the loading skeleton block: Instead of flat `<Skeleton key={i} className="h-[120px] rounded-xl" />`, use `<GlassShimmer key={i} className="h-[120px]" />`.

Replace the stat cards grid container: Swap the outer `<div className="grid ...">` with `<motion.div className="grid ..." variants={containerVariants} initial="hidden" animate="show">`. Wrap each `<StatCard>` in `<motion.div variants={cardVariants}>`.

Use `initial="hidden" animate="show"` (NOT `whileInView`) so animation fires once on component mount. Since DashboardGrid is a client component that conditionally renders after `isLoading` resolves, the animation fires when data arrives. Tab switches do not cause unmount/remount (React preserves the component), so animation does NOT replay on tab switch. Fresh navigation remounts the component, which is expected behavior.

The four StatCards need individual motion.div wrappers. Keep the existing StatCard props exactly as they are.
