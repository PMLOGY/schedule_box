<task type="auto">
  <name>Task 1: Write comprehensive tests for shared utility functions</name>
  <files>
    packages/shared/src/utils/index.test.ts
  </files>
  <action>
Replace the smoke test from Plan 01 with comprehensive tests for ALL exported functions in packages/shared/src/utils/index.ts. Read the source file first to understand exact function signatures and edge cases.

Test each function with the AAA pattern (Arrange-Act-Assert):

1. **generateSlug(text: string): string**
   - Happy path: 'Salon Krasa' -> 'salon-krasa'
   - Czech diacritics: 'Kadeřnictví u Říčky' -> 'kadernictvi-u-ricky'
   - Slovak diacritics: 'Kúpeľné Služby' -> 'kupelne-sluzby'
   - Special characters: 'Hello!!! World???' -> 'hello-world'
   - Leading/trailing hyphens: '---test---' -> 'test'
   - Multiple spaces/separators: 'a   b   c' -> 'a-b-c'
   - Empty string: '' -> ''

2. **formatCurrency(amount, currency?, locale?): string**
   - CZK default: formatCurrency(1500) -> contains '1' and '500' and 'Kc' (Czech locale)
   - CZK with 0 fraction digits: formatCurrency(1500, 'CZK') should have no decimals
   - EUR with 2 fraction digits: formatCurrency(29.99, 'EUR', 'cs-CZ')
   - String amount: formatCurrency('1500.50', 'CZK')
   - Zero amount: formatCurrency(0)
   - Negative amount: formatCurrency(-500)

3. **calculatePagination(total, page, limit): object**
   - Standard: calculatePagination(100, 1, 20) -> { total: 100, page: 1, limit: 20, total_pages: 5 }
   - Partial last page: calculatePagination(101, 1, 20) -> total_pages: 6
   - Zero total: calculatePagination(0, 1, 20) -> total_pages: 1 (default 1, not 0)
   - Single item: calculatePagination(1, 1, 20) -> total_pages: 1

4. **calculateOffset(page, limit): number**
   - Page 1: calculateOffset(1, 20) -> 0
   - Page 3: calculateOffset(3, 20) -> 40
   - Page 1 with limit 50: calculateOffset(1, 50) -> 0

5. **parseStartOfDayUTC(dateStr) / parseEndOfDayUTC(dateStr)**
   - parseStartOfDayUTC('2026-02-15') -> Date at T00:00:00.000Z
   - parseEndOfDayUTC('2026-02-15') -> Date at T23:59:59.999Z
   - Verify they return Date instances
   - Verify the timestamp values are correct

6. **maskSensitive(value, type): string**
   - Email: maskSensitive('john@example.com') -> 'j***@example.com'
   - Email no domain: maskSensitive('invalid') -> '***'
   - Phone: maskSensitive('+420123456789', 'phone') -> '+420***789'
   - Short phone: maskSensitive('12345', 'phone') -> '***'

7. **parseNumericOrNull(value): number | null**
   - Valid: parseNumericOrNull('42') -> 42
   - Float: parseNumericOrNull('3.14') -> 3.14
   - Null: parseNumericOrNull(null) -> null
   - Undefined: parseNumericOrNull(undefined) -> null
   - Empty string: parseNumericOrNull('') -> null
   - NaN string: parseNumericOrNull('abc') -> null
   - Infinity: parseNumericOrNull('Infinity') -> null (Number.isFinite rejects)

8. **omit(obj, keys): object**
   - omit({ a: 1, b: 2, c: 3 }, ['b']) -> { a: 1, c: 3 }
   - omit({ a: 1 }, []) -> { a: 1 }
   - Multiple keys: omit({ a: 1, b: 2, c: 3 }, ['a', 'c']) -> { b: 2 }

9. **pick(obj, keys): object**
   - pick({ a: 1, b: 2, c: 3 }, ['a', 'c']) -> { a: 1, c: 3 }
   - pick({ a: 1 }, ['a', 'b' as any]) -> { a: 1 } (missing key ignored)
   - Empty keys: pick({ a: 1 }, []) -> {}

Use `describe` blocks to group tests by function. Use `it` or `test` with descriptive names: `it('should handle Czech diacritics in generateSlug')`.

**General pattern for Zod schema tests:**
```typescript
describe('schemaName', () => {
  it('validates correct data', () => {
    const result = schema.safeParse(validData);
    expect(result.success).toBe(true);
  });
  it('rejects missing required fields', () => {
    const result = schema.safeParse({});
    expect(result.success).toBe(false);
  });
  it('rejects invalid field values', () => {
    const result = schema.safeParse({ field: 'bad-value' });
    expect(result.success).toBe(false);
  });
});
```
