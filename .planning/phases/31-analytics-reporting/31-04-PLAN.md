  <action>
**1. Create hooks file (use-extended-analytics.ts):**

Follow the pattern of use-revenue-analytics.ts and use-booking-analytics.ts. Use `useQuery` from @tanstack/react-query with `apiClient.get()`. Each hook takes `days: number` param.

Create these hooks:
- `usePaymentMethodAnalytics(days)` -> GET /analytics/payment-methods
- `useTopServicesAnalytics(days)` -> GET /analytics/top-services
- `usePeakHoursAnalytics(days)` -> GET /analytics/peak-hours
- `useEmployeeUtilization(days)` -> GET /analytics/employees
- `useCancellationAnalytics(days)` -> GET /analytics/cancellations
- `useCustomerRetention(days)` -> GET /analytics/customer-retention

All with `staleTime: 5 * 60 * 1000` (5 min cache). Query keys: `['analytics', '{endpoint}', days]`.

Define TypeScript interfaces for each response shape matching the API route returns.

**2. Create payment-method-chart.tsx:**
- Use Recharts `PieChart`, `Pie`, `Cell`, `Tooltip`, `Legend` from 'recharts'
- Props: `data: Array<{ gateway: string, count: number, totalAmount: number, percentage: string }>`
- Color map: comgate=#3B82F6, qrcomat=#10B981, cash=#F59E0B, bank_transfer=#8B5CF6, gift_card=#EC4899
- Use `ChartContainer` wrapper (existing component)
- Label each slice with gateway name (Czech-friendly: "Kartou", "QR", "Hotovost", "Prevodem", "Poukazka")
- Show percentage in tooltip

**3. Create top-services-chart.tsx:**
- Use Recharts `BarChart`, `Bar`, `XAxis`, `YAxis`, `Tooltip`, `CartesianGrid`
- Props: `data: Array<{ serviceName: string, bookingCount: number, totalRevenue: number }>`
- Horizontal BarChart with serviceName on Y-axis, totalRevenue on X-axis
- Color: #3B82F6, hover state via Tooltip
- Use `ChartContainer` wrapper

**4. Create peak-hours-heatmap.tsx:**
- NOT a Recharts component — use a custom CSS grid (7 columns x 24 rows or 24 columns x 7 rows)
- Props: `data: Array<{ dayOfWeek: number, hour: number, count: number }>`
- Convert sparse array to 7x24 grid (fill missing with 0)
- Color scale: white (0) -> light blue -> dark blue (max count)
- Use Tailwind classes. Day labels: Po, Ut, St, Ct, Pa, So, Ne (Czech). Hour labels: 0-23.
- Only show hours 6-22 (business hours) to keep the heatmap compact. Include a note for the user.

**5. Create employee-utilization-chart.tsx:**
- Use Recharts `BarChart` with two grouped bars: bookingCount and occupancyPercent
- Props: `data: Array<{ employeeName: string, bookingCount: number, totalRevenue: number, occupancyPercent: number }>`
- Left Y-axis: employee names, Right Y-axis could be revenue or just bookings
- Use a simple vertical BarChart with employeeName on XAxis
- Bar 1: bookingCount (blue), Bar 2: occupancyPercent (green)
- Tooltip shows all three values

**6. Create cancellation-chart.tsx:**
- Use Recharts `LineChart` with two lines: cancelRate and noShowRate
- Props: `data: Array<{ date: string, cancelRate: number, noShowRate: number }>`
- Format rates as percentages in tooltip
- Use date-fns format for X-axis dates (Czech locale)

**7. Create customer-retention-panel.tsx:**
- NOT a single chart — a panel with three sub-sections:
- Section 1: Repeat booking stats — show repeatRate as a large number with totalCustomers/repeatCustomers below
- Section 2: Churn breakdown — three colored badges: active (green), atRisk (yellow), churned (red) with counts
- Section 3: CLV distribution — simple Recharts BarChart with range buckets on X-axis, count on Y-axis
- Use Card/CardContent from shadcn/ui
- Props match the customer-retention API response shape

All components must be 'use client' directives. All should use `useTranslations` for labels where practical, but hardcoded Czech strings are acceptable for chart labels (the existing charts do this too — see revenue-chart.tsx using "Trzby (Kc)").
