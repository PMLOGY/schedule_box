  <action>
    1. Install required dependencies in apps/web:
       ```bash
       cd apps/web
       pnpm add class-variance-authority clsx tailwind-merge lucide-react
       pnpm add @radix-ui/react-slot @radix-ui/react-label @radix-ui/react-dialog @radix-ui/react-select @radix-ui/react-dropdown-menu @radix-ui/react-avatar @radix-ui/react-separator @radix-ui/react-tooltip @radix-ui/react-scroll-area
       pnpm add react-hook-form @hookform/resolvers
       pnpm add tailwindcss-animate
       ```

    2. Create `apps/web/lib/utils.ts` with cn() utility:
       ```typescript
       import { clsx, type ClassValue } from 'clsx';
       import { twMerge } from 'tailwind-merge';

       export function cn(...inputs: ClassValue[]) {
         return twMerge(clsx(inputs));
       }
       ```

    3. Update `apps/web/tailwind.config.ts` to include:
       - ScheduleBox brand colors as CSS variables (primary #3B82F6 / blue-500, secondary #22C55E / green-500, danger #EF4444, warning #F59E0B)
       - shadcn/ui compatible theme structure using CSS variables (--background, --foreground, --card, --popover, --primary, --secondary, --muted, --accent, --destructive, --border, --input, --ring)
       - Inter font family as default sans
       - Border radius variable (--radius: 0.5rem)
       - tailwindcss-animate plugin
       - Content paths: './app/**/*.{ts,tsx}', './components/**/*.{ts,tsx}', '../../packages/ui/src/**/*.{ts,tsx}'

    4. Update `apps/web/app/globals.css` with:
       - @tailwind base/components/utilities directives
       - @layer base with :root CSS variables for light theme (matching shadcn/ui default + ScheduleBox colors)
       - .dark class CSS variables for dark theme support (future use)
       - body base styles: bg-background, text-foreground, antialiased, font-sans
       - Note: Use HSL values without the hsl() wrapper in variables (shadcn/ui convention): e.g., `--primary: 217 91% 60%;` for #3B82F6

    5. Create `apps/web/components.json` for shadcn/ui CLI configuration:
       ```json
       {
         "$schema": "https://ui.shadcn.com/schema.json",
         "style": "default",
         "rsc": true,
         "tsx": true,
         "tailwind": {
           "config": "tailwind.config.ts",
           "css": "app/globals.css",
           "baseColor": "slate",
           "cssVariables": true
         },
         "aliases": {
           "components": "@/components",
           "utils": "@/lib/utils"
         }
       }
       ```

  <action>
    Create shadcn/ui components following the official shadcn/ui source code patterns. Each component:
    - Uses forwardRef pattern
    - Imports cn() from "@/lib/utils"
    - Uses cva (class-variance-authority) for variants where applicable
    - Uses Radix UI primitives underneath
    - Exports named components (not default exports)

    Create these components in apps/web/components/ui/:

    1. **button.tsx** — Button with variants: default, destructive, outline, secondary, ghost, link. Sizes: default, sm, lg, icon. Uses cva for variant styles. Include isLoading prop that shows a spinner and disables the button.

    2. **input.tsx** — Input with focus ring, error state styling. Accepts className prop merged via cn().

    3. **label.tsx** — Label using @radix-ui/react-label with peer error styling.

    4. **card.tsx** — Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter as separate named exports.

    5. **badge.tsx** — Badge with variants: default, secondary, destructive, outline. Uses cva.

    6. **avatar.tsx** — Avatar, AvatarImage, AvatarFallback using @radix-ui/react-avatar.

    7. **dialog.tsx** — Dialog, DialogTrigger, DialogContent, DialogHeader, DialogTitle, DialogDescription, DialogFooter using @radix-ui/react-dialog. Include overlay with backdrop blur.

    8. **select.tsx** — Select, SelectTrigger, SelectContent, SelectItem, SelectValue using @radix-ui/react-select. Include chevron icon from lucide-react.

    9. **dropdown-menu.tsx** — DropdownMenu, DropdownMenuTrigger, DropdownMenuContent, DropdownMenuItem, DropdownMenuSeparator, DropdownMenuLabel using @radix-ui/react-dropdown-menu.

    10. **skeleton.tsx** — Skeleton component with animate-pulse and rounded styling.

    11. **separator.tsx** — Separator using @radix-ui/react-separator, supports horizontal and vertical.

    12. **sheet.tsx** — Sheet (slide-over panel) using @radix-ui/react-dialog. Supports side: top, right, bottom, left. Used for mobile navigation.

    13. **tooltip.tsx** — Tooltip, TooltipTrigger, TooltipContent, TooltipProvider using @radix-ui/react-tooltip.

    14. **scroll-area.tsx** — ScrollArea, ScrollBar using @radix-ui/react-scroll-area.

    15. **form.tsx** — Form integration with react-hook-form: Form, FormField, FormItem, FormLabel, FormControl, FormDescription, FormMessage. Uses React Context to pass form field state. useFormField hook for accessing field error state.

    16. **table.tsx** — Table, TableHeader, TableBody, TableFooter, TableHead, TableRow, TableCell, TableCaption as simple HTML wrappers with Tailwind styling.

    Follow the exact shadcn/ui source code patterns (available at https://ui.shadcn.com/docs/components/). Every component should use 'use client' directive only if it uses hooks or browser APIs. Pure styling wrappers (Card, Table, Skeleton) should NOT have 'use client'.
