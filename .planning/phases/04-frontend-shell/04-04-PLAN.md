    1. Create `apps/web/components/auth/register-form.tsx` (Client Component):
       - 'use client' directive
       - Zod schema: firstName (min 2), lastName (min 2), email (email), password (min 12, must match complexity regex from Phase 3: uppercase + lowercase + number + special), confirmPassword (must match password), companyName (min 2)
       - Use React Hook Form with zodResolver
       - Use shadcn/ui Form components (FormField, FormItem, FormLabel, FormControl, FormMessage)
       - Use useTranslations('auth.register') for all text
       - On submit: POST to /api/v1/auth/register with { email, password, firstName, lastName, companyName }
       - On success: show success message + redirect to login
       - On error: show server error message
       - Link: "Already have an account?" links to /login
       - Password strength indicator (optional): simple visual bar showing weak/medium/strong

    2. Create `apps/web/app/(auth)/register/page.tsx`:
       - Server Component, renders RegisterForm
       - Metadata: { title: 'Register | ScheduleBox' }

    3. Create `apps/web/components/auth/forgot-password-form.tsx` (Client Component):
       - Zod schema: email (email)
       - On submit: POST to /api/v1/auth/forgot-password with { email }
       - Always show success message (API never reveals if email exists — security)
       - Use useTranslations('auth.forgotPassword')
       - Link: "Back to login" links to /login

    4. Create `apps/web/app/(auth)/forgot-password/page.tsx`:
       - Server Component, renders ForgotPasswordForm
       - Metadata: { title: 'Forgot Password | ScheduleBox' }

    5. Create `apps/web/components/auth/reset-password-form.tsx` (Client Component):
       - Read token from URL search params (e.g., /reset-password?token=xyz)
       - Zod schema: password (min 12, complexity), confirmPassword (must match)
       - On submit: POST to /api/v1/auth/reset-password with { token, newPassword }
       - On success: show success message + link to login
       - On error: show invalid/expired token error
       - Use useTranslations('auth.resetPassword')

    6. Create `apps/web/app/(auth)/reset-password/page.tsx`:
       - Server Component, renders ResetPasswordForm
       - Metadata: { title: 'Reset Password | ScheduleBox' }
