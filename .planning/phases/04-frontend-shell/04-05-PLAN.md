  <action>
    1. Update `apps/web/app/layout.tsx` (Server Component):
       - Import Inter font from 'next/font/google'
       - Apply Inter font className to <html>
       - Wrap {children} with Providers from './providers'
       - Add <Toaster /> from sonner (install sonner if not already: `pnpm add sonner`)
       - Keep lang="cs" on <html>
       - Include NextIntlClientProvider for i18n support (import getMessages, getLocale from 'next-intl/server')
       - Metadata: title template '%s | ScheduleBox', default 'ScheduleBox'

    2. Create `apps/web/hooks/use-auth.ts`:
       - Custom hook useAuth() that:
         - Gets user, isAuthenticated, logout from useAuthStore
         - On mount, if !isAuthenticated and window.location is a dashboard route, redirect to /login via router.push('/login')
         - Returns { user, isAuthenticated, logout, isLoading }
         - isLoading is true during initial hydration check (to prevent flash of login redirect)
       - This is a Client-side hook (used in Client Components)

    3. Create `apps/web/components/layout/sidebar.tsx` (Client Component):
       - 'use client'
       - Import useUIStore for sidebarCollapsed state
       - Import useAuthStore for user role
       - Import useTranslations('nav') for navigation labels
       - Import icons from lucide-react: LayoutDashboard, Calendar, BookOpen, Users, Scissors, UserCog, BarChart3, Megaphone, Settings, HelpCircle, ChevronLeft, ChevronRight
       - Navigation items array with: { key (translation key), href, icon, roles: string[] }
         - dashboard: '/' — all roles
         - calendar: '/calendar' — owner, manager, employee
         - bookings: '/bookings' — owner, manager, employee
         - customers: '/customers' — owner, manager, employee
         - services: '/services' — owner, manager
         - employees: '/employees' — owner, manager
         - analytics: '/analytics' — owner
         - marketing: '/marketing' — owner, manager
         - settings: '/settings' — owner
       - Filter navigation items by current user's role
       - Sidebar width: 256px (w-64) expanded, 64px (w-16) collapsed
       - Toggle button at bottom with ChevronLeft/ChevronRight icon
       - Active state: highlight current route (usePathname from 'next/navigation')
       - Tooltip on collapsed items showing the full label
       - ScheduleBox logo/name at top (full in expanded, icon-only in collapsed)
       - Smooth width transition (transition-all duration-300)

    4. Create `apps/web/components/layout/user-menu.tsx` (Client Component):
       - 'use client'
       - Uses DropdownMenu from shadcn/ui
       - Trigger: Avatar with user initials + user name (hidden when sidebar collapsed)
       - Menu items: Profile (/settings/profile), Settings (/settings), Logout
       - Logout calls useAuthStore.logout() then router.push('/login')
       - Company name displayed below user name in trigger area

    5. Create `apps/web/app/(dashboard)/layout.tsx` (Client Component wrapper):
       - 'use client' directive (because it uses useAuth hook for auth guard)
       - Import useAuth hook — if not authenticated, show loading spinner or redirect
       - Layout structure:
         ```
         <div className="flex h-screen">
           <Sidebar />
           <div className="flex flex-1 flex-col overflow-hidden">
             <Header />
             <main className="flex-1 overflow-y-auto p-6">
               <div className="mx-auto max-w-7xl">
                 {children}
               </div>
             </main>
           </div>
         </div>
         ```
       - On mobile (< md): sidebar hidden, mobile nav via Sheet
       - Max content width: 1280px (max-w-7xl ~ 80rem)
