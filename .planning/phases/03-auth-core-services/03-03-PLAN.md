    Create `apps/web/lib/middleware/auth.ts`:
    - Import { NextRequest } from 'next/server'
    - Import { verifyJWT, JWTPayload } from '@/lib/auth/jwt'
    - Import { UnauthorizedError } from '@schedulebox/shared'
    - authenticateRequest(req: NextRequest): Promise<JWTPayload>
      - Extract Authorization header
      - If missing or doesn't start with 'Bearer ', throw UnauthorizedError('Missing or invalid authorization header')
      - Extract token (substring after 'Bearer ')
      - Call verifyJWT(token) and return payload
      - On error, re-throw if UnauthorizedError, otherwise throw new UnauthorizedError('Authentication failed')

    Create `apps/web/lib/middleware/rbac.ts`:
    - Import { JWTPayload } from '@/lib/auth/jwt'
    - Import { ForbiddenError } from '@schedulebox/shared'
    - Define PERMISSIONS constant object with all 23 permission strings from documentation (lines 1010-1033):
      BOOKINGS_CREATE: 'bookings.create', BOOKINGS_READ: 'bookings.read', BOOKINGS_UPDATE: 'bookings.update', BOOKINGS_DELETE: 'bookings.delete',
      CUSTOMERS_CREATE: 'customers.create', CUSTOMERS_READ: 'customers.read', CUSTOMERS_UPDATE: 'customers.update', CUSTOMERS_DELETE: 'customers.delete',
      SERVICES_CREATE: 'services.create', SERVICES_READ: 'services.read', SERVICES_UPDATE: 'services.update', SERVICES_DELETE: 'services.delete',
      EMPLOYEES_MANAGE: 'employees.manage', RESOURCES_MANAGE: 'resources.manage',
      PAYMENTS_READ: 'payments.read', PAYMENTS_REFUND: 'payments.refund',
      REPORTS_READ: 'reports.read', SETTINGS_MANAGE: 'settings.manage',
      LOYALTY_MANAGE: 'loyalty.manage', COUPONS_MANAGE: 'coupons.manage',
      MARKETPLACE_MANAGE: 'marketplace.manage', AI_USE: 'ai.use', WHITELABEL_MANAGE: 'whitelabel.manage'
    - checkPermissions(user: JWTPayload, requiredPermissions: string[]): void
      - If requiredPermissions is empty, return (no check needed)
      - Check if user.permissions includes ALL required permissions
      - If any missing, throw ForbiddenError('Insufficient permissions')

    Create `apps/web/lib/middleware/route-handler.ts`:
    - Import all needed types and functions
    - Define RouteHandlerOptions<TBody, TParams> generic type:
      - bodySchema?: z.ZodSchema<TBody>
      - paramsSchema?: z.ZodSchema<TParams>
      - requiresAuth?: boolean (default true)
      - requiredPermissions?: string[]
      - handler: (context: RouteHandlerContext<TBody, TParams>) => Promise<NextResponse>
    - Define RouteHandlerContext<TBody, TParams>:
      - req: NextRequest
      - body: TBody (undefined if no bodySchema)
      - params: TParams (undefined if no paramsSchema)
      - user: JWTPayload (undefined if requiresAuth=false)
    - createRouteHandler<TBody, TParams>(options: RouteHandlerOptions<TBody, TParams>):
      Returns async function (req: NextRequest, context?: { params: Promise<any> }) => Promise<NextResponse>
      Implementation order:
      1. Wrap everything in try/catch, catch calls handleRouteError from utils/errors.ts
      2. If requiresAuth (default true), call authenticateRequest(req) -> user
      3. If requiredPermissions provided and user exists, call checkPermissions(user, requiredPermissions)
      4. If bodySchema provided, call validateBody(bodySchema, req) -> body
      5. If paramsSchema provided and context?.params exists, await context.params then validateParams(paramsSchema, resolved params) -> params
      6. Call options.handler({ req, body, params, user })
      7. Return the handler's NextResponse directly (handler is responsible for formatting response)

    IMPORTANT for Next.js 14 App Router: The params in route handlers are now a Promise that must be awaited. The signature is (req: NextRequest, { params }: { params: Promise<{ id: string }> }). Make sure createRouteHandler handles this correctly by awaiting params before validation.

    Create `apps/web/lib/db/tenant-scope.ts`:

    findCompanyId(userUuid: string): Promise<{ companyId: number; companyUuid: string }>

    - If no user found or no companyId, throw UnauthorizedError('User has no associated company')
    - Query companies WHERE id = companyId, select uuid
    - Return { companyId (internal SERIAL), companyUuid (UUID string) }
    - This is THE canonical function used by all CRUD endpoints to resolve company scope from JWT

    Export findCompanyId as the primary export. All Wave 3 CRUD plans will import and use this function.
