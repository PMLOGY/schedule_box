<task type="auto">
  <name>Task 1: Create industry config modules + Zod schemas + useIndustryLabels hook</name>
  <files>
    apps/web/lib/industry/industry-labels.ts

  <action>
1. Create `apps/web/lib/industry/industry-labels.ts`:
   - Export `IndustryLabels` interface with fields: `customer`, `booking`, `service`, `newBooking`, `customerSearch`, `bookingDetail`, `serviceList`.
   - Define `DEFAULT_LABELS` with Czech generic names: Zakaznik, Rezervace, Sluzba, Nova rezervace, Hledat zakaznika, Detail rezervace, Seznam sluzeb.
   - Define `INDUSTRY_LABEL_MAP: Record<string, Partial<IndustryLabels>>` with:
     - `medical_clinic`: Pacient, Termin, Vysetreni, Novy termin, Hledat pacienta, Detail terminu, Seznam vysetreni
     - `auto_service`: Vozidlo, Zakazka, Servis, Nova zakazka, Hledat vozidlo, Detail zakazky, Seznam servisu
   - Export `getIndustryLabels(industryType: string): IndustryLabels` that spreads defaults with overrides.
   - NOTE: These are NOT i18n translations — they are business logic labels. Do NOT put them in messages/cs.json.

2. Create `apps/web/lib/industry/industry-fields.ts`:
   - Export `VerticalField` interface: `{ key: string; label: string; placeholder: string; required: boolean; maxLength: number }`.
   - Export `VERTICAL_FIELDS: Record<string, VerticalField[]>` with:
     - `medical_clinic`: [birth_number (Rodne cislo, NNNNNN/NNNN, optional, 20), insurance_provider (Pojistovna, VZP/CPZP/OZP..., optional, 100)]
     - `auto_service`: [license_plate (SPZ, 1A2 3456, optional, 20), vin (VIN, 17 chars, optional, 17)]
   - Export Zod schemas for booking metadata validation:
     - `medicalMetadataSchema = z.object({ industry_type: z.literal('medical_clinic'), birth_number: z.string().max(20).optional(), insurance_provider: z.string().max(100).optional() })`
     - `autoMetadataSchema = z.object({ industry_type: z.literal('auto_service'), license_plate: z.string().max(20).optional(), vin: z.string().max(17).optional() })`
     - `bookingMetadataSchema = z.discriminatedUnion('industry_type', [medicalMetadataSchema, autoMetadataSchema]).nullable().optional()`
   - Use proper Czech characters with diacritics in labels and placeholders. Check industry-templates.ts for Czech conventions.

3. Create `apps/web/lib/industry/industry-ai-defaults.ts`:
   - Export `IndustryAiConfig` interface: `{ upselling_enabled: boolean; capacity_mode: 'individual' | 'group' | 'standard' }`.
   - Define `INDUSTRY_AI_DEFAULTS: Record<string, IndustryAiConfig>`:
     - `medical_clinic`: upselling_enabled=false, capacity_mode='individual'
     - `fitness_gym`: upselling_enabled=true, capacity_mode='group'
     - `yoga_pilates`: upselling_enabled=true, capacity_mode='group'
   - Define `DEFAULT_AI_CONFIG`: upselling_enabled=true, capacity_mode='standard'.
   - Export `getIndustryAiDefaults(industryType: string): IndustryAiConfig`.

4. Create `apps/web/hooks/use-industry-labels.ts`:
   - Import `useCompanySettingsQuery` from `@/hooks/use-settings-query`.
   - Import `getIndustryLabels` from `@/lib/industry/industry-labels`.
   - Import `useMemo` from React.
   - Export `useIndustryLabels()` hook that:
     - Gets company settings via `useCompanySettingsQuery()`
     - Returns `useMemo(() => getIndustryLabels(data?.industry_type ?? 'general'), [data?.industry_type])`
   - This is a simple thin hook — the logic lives in industry-labels.ts.
  </action>
