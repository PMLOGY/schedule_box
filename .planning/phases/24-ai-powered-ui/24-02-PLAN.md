    Create `apps/web/app/api/v1/ai/insights/route.ts`:
    - Use the `createRouteHandler` pattern (same as existing bookings/analytics routes)
    - Require auth with `PERMISSIONS.BOOKINGS_READ`
    - Find company ID via `findCompanyId(user.sub)` for tenant isolation
    - Execute TWO queries:

      Query 1 -- Today's high-risk bookings:
      ```sql
      SELECT b.uuid AS booking_id,
             c.name AS customer_name,
             s.name AS service_name,
             b.start_time,
             b.no_show_probability
      FROM bookings b
      JOIN customers c ON c.id = b.customer_id
      JOIN services s ON s.id = b.service_id
      WHERE b.company_id = $companyId
        AND b.status IN ('pending', 'confirmed')
        AND b.start_time >= NOW()
        AND b.start_time < NOW() + INTERVAL '24 hours'
        AND b.no_show_probability IS NOT NULL
        AND b.no_show_probability >= 0.30
      ORDER BY b.no_show_probability DESC
      LIMIT 10
      ```
      Use drizzle-orm `sql` template for this raw query (same pattern as the internal training routes in Plan 23-01). Import `db` from `@schedulebox/database` and `sql` from `drizzle-orm`.

      Query 2 -- Total company bookings count (for AI activation threshold):
      ```sql
      SELECT COUNT(*) AS total_bookings
      FROM bookings
      WHERE company_id = $companyId
        AND status IN ('completed', 'confirmed', 'no_show', 'pending')
      ```

    - Also count today's total upcoming bookings (use a third simple query or extend Query 1):
      ```sql
      SELECT COUNT(*) AS today_total
      FROM bookings
      WHERE company_id = $companyId
        AND status IN ('pending', 'confirmed')
        AND start_time >= NOW()
        AND start_time < NOW() + INTERVAL '24 hours'
      ```

    - Build response object:
      ```typescript
      {
        highRiskBookings: Array<{
          bookingId: string;
          customerName: string;
          serviceName: string;
          startTime: string;
          noShowProbability: number;
          riskLevel: 'medium' | 'high';
        }>;
        totalTodayBookings: number;
        highRiskCount: number;
        aiActive: boolean;           // true if totalCompanyBookings >= 10
        totalCompanyBookings: number; // for progress indicator
        suggestions: string[];       // generated server-side based on data
      }
      ```
    - Compute `riskLevel` from probability: >= 0.50 is 'high', 0.30-0.49 is 'medium'
    - Generate suggestions array based on data:
      - If highRiskCount > 0: "Doporucujeme odeslat SMS pripominky {count} zakaznikum s vysokym rizikem nedostaveni"
      - If highRiskCount > 3: "Zvazite dvojitou konfirmaci pro zakaziky s rizikem nad 50%"
      - If totalTodayBookings === 0: "Na dnes nemáte zadne nadchazejici rezervace"
    - Return via `successResponse(result)`

    Create `apps/web/hooks/use-ai-insights-query.ts`:
    - Import `useQuery` from `@tanstack/react-query`
    - Import `apiClient` from `@/lib/api-client`
    - Define the `AiInsightsData` TypeScript interface matching the API response shape above
    - Export `useAiInsightsQuery()` hook:
      ```typescript
      export function useAiInsightsQuery() {
        return useQuery({
          queryKey: ['ai', 'insights'],
          queryFn: async () => {
            return apiClient.get<AiInsightsData>('/ai/insights');
          },
          staleTime: 2 * 60 * 1000, // 2 minutes - insights are somewhat stable
          refetchInterval: 5 * 60 * 1000, // Auto-refresh every 5 minutes on dashboard
        });
      }
      ```
    - Export the `AiInsightsData` type for use by components

      </CardHeader>
      <CardContent className="space-y-4">
        {/* High-risk bookings summary */}
        {data.highRiskCount > 0 ? (
          <>
            <div className="flex items-center gap-2 text-sm font-medium text-red-700">
              <AlertTriangle className="h-4 w-4" />
              {t('ai.insights.highRiskToday', { count: data.highRiskCount })}
            </div>
            {/* List of high-risk bookings (max 5 shown) */}
            <div className="space-y-2">
              {data.highRiskBookings.slice(0, 5).map((booking) => (
                <div key={booking.bookingId} className="flex items-center justify-between rounded-lg border p-3 text-sm">
                  <div className="space-y-0.5">
                    <div className="font-medium">{booking.customerName}</div>
                    <div className="text-muted-foreground">{booking.serviceName} - {format(new Date(booking.startTime), 'HH:mm')}</div>
                  </div>
                  <NoShowRiskBadge probability={booking.noShowProbability} />
                </div>
              ))}
            </div>
          </>

          <div className="text-sm text-muted-foreground">
            {t('ai.insights.noHighRisk')}
          </div>
        )}

        {/* Suggestions */}
        {data.suggestions.length > 0 && (
          <div className="rounded-lg bg-muted/50 p-3 space-y-1">
            {data.suggestions.map((suggestion, i) => (
              <p key={i} className="text-sm text-muted-foreground">{suggestion}</p>
            ))}
          </div>
        )}

        {/* Link to bookings */}
        <Link href="/bookings" className="flex items-center gap-1 text-sm text-primary hover:underline">
          {t('ai.insights.viewAll')} <ArrowRight className="h-3 w-3" />
        </Link>
      </CardContent>
    </Card>
