1. **"Last updated" indicator:**
   - Add state: `const [lastUpdated, setLastUpdated] = useState<Date>(new Date());`

   - Add a 1-second interval to force re-render the relative time display: `const [, setTick] = useState(0); useEffect(() => { const i = setInterval(() => setTick(t => t + 1), 1000); return () => clearInterval(i); }, []);`

2. **New booking glow animation:**
   - Track previous booking IDs in a ref: `const prevIdsRef = useRef<Set<string>>(new Set());`
   - Track new booking IDs: `const [newBookingIds, setNewBookingIds] = useState<Set<string>>(new Set());`
   - On data change, detect new IDs:
     ```
     useEffect(() => {
       if (!data?.data) return;
       const currentIds = new Set(data.data.map(b => b.id));
       if (prevIdsRef.current.size > 0) {
         const newIds = [...currentIds].filter(id => !prevIdsRef.current.has(id));
         if (newIds.length > 0) {
           setNewBookingIds(new Set(newIds));
           setTimeout(() => setNewBookingIds(new Set()), 4000);
         }
       }
       prevIdsRef.current = currentIds;
     }, [data]);
     ```
   - In the table row rendering (or wrapper around data-table rows), apply a CSS class or inline style for glow:
     - If data-table supports custom row className/style, pass a function: `rowClassName={(row) => newBookingIds.has(row.id) ? 'animate-glow-blue' : ''}`
     - If data-table does not support per-row styling, use `motion` library: wrap row content in `<motion.div>` with `animate` prop for box-shadow glow when `newBookingIds.has(row.id)`
     - Glow style: `box-shadow: 0 0 12px rgba(0, 87, 255, 0.5)` fading to `0 0 0px` over 4 seconds
   - Import `motion` from the existing motion library (framer-motion or "motion" package, check existing imports in the codebase)
   - Add a CSS keyframe animation as a Tailwind class if simpler: `@keyframes glow-blue { 0% { box-shadow: 0 0 12px #0057FF88; } 100% { box-shadow: 0 0 0px #0057FF00; } }` in globals.css or use inline motion animate.
