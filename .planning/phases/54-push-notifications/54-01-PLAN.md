1. Install `web-push` package: `pnpm --filter @schedulebox/web add web-push` and `pnpm --filter @schedulebox/web add -D @types/web-push`

2. Create `packages/database/src/schema/push-subscriptions.ts`:
   - Table `push_subscriptions` with columns:
     - `id` serial PK
     - `userId` integer NOT NULL references users.id ON DELETE CASCADE
     - `endpoint` text NOT NULL (the push service URL)
     - `keysP256dh` text NOT NULL
     - `keysAuth` text NOT NULL
     - `userAgent` varchar(500) nullable (for device identification in UI)
     - `createdAt` timestamp with timezone default now
   - Add unique constraint on (userId, endpoint) to prevent duplicate subscriptions
   - Add index on userId for fast lookup
   - RLS note: userId column provides tenant scoping (user can only manage own subscriptions)

3. Add `export * from './push-subscriptions';` to `packages/database/src/schema/index.ts`

4. Run `pnpm --filter @schedulebox/database db:push` to sync schema to Neon DB

5. Create `apps/web/lib/push/web-push-service.ts`:
   - Import `webpush` from 'web-push'
   - On module load, call `webpush.setVapidDetails('mailto:admin@schedulebox.cz', process.env.VAPID_PUBLIC_KEY!, process.env.VAPID_PRIVATE_KEY!)`
   - Export `async function sendPushNotification(subscription: { endpoint: string; keys: { p256dh: string; auth: string } }, payload: { title: string; body: string; url?: string; icon?: string }): Promise<boolean>` — wraps webpush.sendNotification, returns true on success, false on 410 Gone (subscription expired — caller should delete it), throws on other errors
   - Export `async function sendPushToUser(userId: number, payload: { title: string; body: string; url?: string }): Promise<{ sent: number; failed: number; expired: number }>` — queries all push_subscriptions for userId, sends to each, deletes expired (410) subscriptions, returns counts
   - Default icon: '/icons/icon-192x192.png'

6. Create `apps/web/app/api/v1/push/subscribe/route.ts`:
   - POST endpoint, requiresAuth: true
   - Body schema: `{ endpoint: string (url), keys: { p256dh: string, auth: string }, userAgent?: string }`
   - Upsert into push_subscriptions (ON CONFLICT userId+endpoint DO UPDATE keysP256dh, keysAuth, userAgent)
   - Return 201 with { message: 'Subscribed' }

7. Create `apps/web/app/api/v1/push/unsubscribe/route.ts`:
   - POST endpoint, requiresAuth: true
   - Body schema: `{ endpoint: string (url) }`
   - Delete from push_subscriptions WHERE userId = current AND endpoint = body.endpoint
   - Return 200 with { message: 'Unsubscribed' }

8. Create `apps/web/app/api/v1/push/vapid-key/route.ts`:
   - GET endpoint, requiresAuth: false (public — needed by service worker registration)
   - Return `{ publicKey: process.env.VAPID_PUBLIC_KEY }` or 500 if not configured

9. Create `apps/web/app/api/v1/push/test/route.ts`:
   - POST endpoint, requiresAuth: true
   - Gets current userId via findCompanyId
   - Calls sendPushToUser(userId, { title: 'ScheduleBox', body: 'Push notifikace funguje!', url: '/dashboard' })
   - Returns result counts

10. The old `/api/v1/webhooks/push/register/route.ts` can remain for backward compatibility but is now superseded by the new `/api/v1/push/subscribe` endpoint.

1. Create `apps/web/public/sw.js` (plain JavaScript, not TypeScript — service workers must be served as-is):
   ```javascript
   // ScheduleBox Push Notification Service Worker
   self.addEventListener('install', (event) => {
     self.skipWaiting();
   });

   self.addEventListener('activate', (event) => {
     event.waitUntil(clients.claim());
   });

   self.addEventListener('push', (event) => {
     if (!event.data) return;
     const data = event.data.json();
     const title = data.title || 'ScheduleBox';
     const options = {
       body: data.body || '',
       icon: data.icon || '/icons/icon-192x192.png',
       badge: '/icons/icon-72x72.png',
       data: { url: data.url || '/' },
       tag: data.tag || 'schedulebox-notification',
       renotify: true,
     };
     event.waitUntil(self.registration.showNotification(title, options));
   });

   self.addEventListener('notificationclick', (event) => {
     event.notification.close();
     const url = event.notification.data?.url || '/';
     event.waitUntil(
       clients.matchAll({ type: 'window', includeUncontrolled: true }).then((clientList) => {
         for (const client of clientList) {
           if (client.url.includes(url) && 'focus' in client) return client.focus();
         }
         return clients.openWindow(url);
       })
     );
   });
   ```
