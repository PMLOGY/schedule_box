1. Create `apps/web/components/push/push-settings-card.tsx`:
   - 'use client' component
   - Uses shadcn Card, CardHeader, CardTitle, CardDescription, CardContent, Switch, Button
   - Uses lucide-react Bell, BellOff, Send icons
   - State: `isSubscribed` boolean, `isLoading` boolean, `isSupported` boolean
   - On mount:
     a. Check `'serviceWorker' in navigator && 'PushManager' in window` -> set isSupported
     b. If supported, get current SW registration via `navigator.serviceWorker.ready`
     c. Check `registration.pushManager.getSubscription()` to see if already subscribed -> set isSubscribed
   - Toggle handler (subscribe):
     a. Fetch VAPID public key from `/api/v1/push/vapid-key`
     b. Convert VAPID key to Uint8Array (urlBase64ToUint8Array helper — standard conversion function, include inline)
     c. Call `registration.pushManager.subscribe({ userVisibleOnly: true, applicationServerKey })` — this triggers the browser permission prompt
     d. POST the subscription to `/api/v1/push/subscribe` with `{ endpoint, keys: { p256dh, auth }, userAgent: navigator.userAgent }`
     e. Toast success: 'Push notifikace zapnuty'
   - Toggle handler (unsubscribe):
     a. Get current subscription via `registration.pushManager.getSubscription()`
     b. Call `subscription.unsubscribe()`
     c. POST to `/api/v1/push/unsubscribe` with `{ endpoint: subscription.endpoint }`
     d. Toast success: 'Push notifikace vypnuty'
   - "Send test" button: POST to `/api/v1/push/test`, toast result
   - If not supported, show info message: "Tento prohlizec nepodporuje push notifikace"
   - All text in Czech (this is a CZ-market app)
