    **Step 2: Create visual regression test** (`apps/web/e2e/tests/embed-widget-visual.spec.ts`)

    Write a Playwright test that:
    1. Navigates to `/embed/test-company` (the seeded demo company slug from dev data — check `dev-quickstart.md` or seed data for the actual slug; if the slug is unknown, use a parameterized approach that reads from env or defaults to `test-company`)
    2. Waits for the widget content to load (wait for `.space-y-3` container or the "Powered by ScheduleBox" footer text)
    3. Takes a full-page screenshot as baseline: `await expect(page).toHaveScreenshot('embed-widget-light.png', { fullPage: true })`

    Also test dark theme variant:
    1. Navigate to `/embed/test-company?theme=dark`
    2. Wait for content
    3. Screenshot: `await expect(page).toHaveScreenshot('embed-widget-dark.png', { fullPage: true })`

    Test structure:
    ```typescript
    import { test, expect } from '@playwright/test';

    const EMBED_SLUG = process.env.EMBED_TEST_SLUG || 'test-company';

    test.describe('Embed Widget Visual Regression', () => {
      test('light theme baseline', async ({ page }) => {
        await page.goto(`/embed/${EMBED_SLUG}?theme=light&locale=cs`);
        await page.waitForSelector('text=Powered by ScheduleBox');
        await expect(page).toHaveScreenshot('embed-widget-light.png', {
          fullPage: true,
        });
      });

      test('dark theme baseline', async ({ page }) => {
        await page.goto(`/embed/${EMBED_SLUG}?theme=dark&locale=cs`);
        await page.waitForSelector('text=Powered by ScheduleBox');
        await expect(page).toHaveScreenshot('embed-widget-dark.png', {
          fullPage: true,
        });
      });

      test('widget with no services shows empty state', async ({ page }) => {
        await page.goto(`/embed/nonexistent-slug-12345`);
        await page.waitForSelector('text=Company not found');
        await expect(page).toHaveScreenshot('embed-widget-not-found.png', {
          fullPage: true,
        });
      });
    });
    ```

    Important: On first run, Playwright creates the baseline screenshots automatically. These go into `apps/web/e2e/tests/embed-widget-visual.spec.ts-snapshots/`. Future runs compare against these baselines.

    Do NOT modify any existing test files. Only add new files and extend the config.
