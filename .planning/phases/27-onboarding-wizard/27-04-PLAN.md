    Create `apps/web/app/api/v1/onboarding/apply-template/route.ts`:
    - Use `createRouteHandler` pattern (requires auth, owner permission)
    - **POST handler:**
      - Accept body: `{ industry_type: string }` — validate with z.object({ industry_type: z.string() })
      - Get companyId from `findCompanyId(user.sub)`
      - Look up template: `getTemplateByIndustry(body.industry_type)` — return 400 if not found
      - In a transaction:
        1. Create all services from the template:
           ```typescript
           for (const svc of template.services) {
             await tx.insert(services).values({
               companyId,
               name: svc.name,
               description: svc.description,
               durationMinutes: svc.durationMinutes,
               bufferAfterMinutes: svc.bufferAfterMinutes,
               price: svc.price.toString(),
               maxCapacity: svc.maxCapacity,
               onlineBookingEnabled: true,
               color: svc.color,
             });
           }
           ```
        2. Replace working hours (delete existing company-level, insert new):
           ```typescript
           await tx.delete(workingHours).where(
             and(eq(workingHours.companyId, companyId), isNull(workingHours.employeeId))
           );
           for (const wh of template.workingHours) {
             if (wh.isActive) {
               await tx.insert(workingHours).values({
                 companyId,
                 employeeId: null,
                 dayOfWeek: wh.dayOfWeek,
                 startTime: wh.startTime,
                 endTime: wh.endTime,
                 isActive: true,
               });
             }
           }
           ```
        3. Update company industry_type:
           ```typescript
           await tx.update(companies).set({ industryType: body.industry_type }).where(eq(companies.id, companyId));
           ```
      - Return: `{ services_created: N, working_hours_set: true, industry_type: body.industry_type }`
