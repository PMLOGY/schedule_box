  <action>
    Modify `services/ai/scripts/train_no_show.py` with these specific changes:

    1. **Remove `use_label_encoder=False`** from the `XGBClassifier()` constructor (line ~205). This parameter was removed in XGBoost 3.x and will cause a TypeError. Simply delete the line — label encoding is disabled by default in modern XGBoost.

    2. **Add `write_meta_sidecar()` function** that writes a `.meta.json` file alongside the model .joblib file:
       ```python
       def write_meta_sidecar(model_path: str, model_name: str, metrics: dict, features: list) -> None:
           import sklearn
           import xgboost as xgb
           meta = {
               "model_name": model_name,
               "model_version": "v1.0.0",
               "trained_at": datetime.now(timezone.utc).isoformat(),
               "sklearn_version": sklearn.__version__,
               "xgboost_version": xgb.__version__,
               "features": features,
               "metrics": metrics,
           }
           meta_path = model_path.replace(".joblib", ".meta.json")
           with open(meta_path, "w") as f:
               json.dump(meta, f, indent=2)
           logger.info(f"Version sidecar written to {meta_path}")
       ```

    3. **Call `write_meta_sidecar()`** after `joblib.dump(model, model_path)` in `train_model()`:
       ```python
       write_meta_sidecar(model_path, "no_show_predictor", avg_metrics, FEATURE_COLUMNS)
       ```

    4. **Add API key header** to the httpx.get() call in `load_training_data()`:
       ```python
       api_key = os.environ.get("AI_SERVICE_API_KEY", "")

           f"{api_url}/api/internal/features/training/no-show",
           headers=headers,
           timeout=30.0,
       )
       ```

    5. **Read API URL from env var** — change the default `api_url` parameter to check `SCHEDULEBOX_API_URL` env var:

       ```python
       # NEW (correct):
       from prophet.serialize import model_to_json

       ```
