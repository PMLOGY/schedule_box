    **Add Prophet warmup to `services/ai/app/services/model_loader.py`:**
    - Add a module-level flag: `_prophet_warmed_up: bool = False`
    - Add function:
      ```python
      async def warmup_prophet() -> None:
          """Run a throwaway prediction to warm up Prophet's Stan backend."""
          global _prophet_warmed_up
          capacity_model = _models.get("capacity")
          if capacity_model and capacity_model.model:
              try:
                  import asyncio
                  loop = asyncio.get_event_loop()
                  await loop.run_in_executor(None, capacity_model.forecast, 1)
                  _prophet_warmed_up = True
                  logger.info("Prophet warmup prediction complete")
              except Exception as e:
                  logger.warning(f"Prophet warmup failed (non-fatal): {e}")
                  _prophet_warmed_up = True  # Still mark as done — don't block health check forever
          else:
              _prophet_warmed_up = True  # No Prophet model loaded — nothing to warm up
              logger.info("Prophet warmup skipped (no capacity model loaded)")

      def is_prophet_warmed_up() -> bool:
          return _prophet_warmed_up
      ```
    - Export `warmup_prophet` and `is_prophet_warmed_up`

    **Modify `services/ai/app/main.py` startup_event:**
    - Import `warmup_prophet` from model_loader
    - After `await load_models()`, call `await warmup_prophet()`
    - The startup sequence becomes: load models -> validate versions -> warmup Prophet -> ready

    **Modify `services/ai/app/routers/health.py` health_check:**
    - Import `is_prophet_warmed_up` from model_loader
    - Return 503 if either `not models_loaded` or `not is_prophet_warmed_up()`
    - Add `prophet_warmed_up` field to the response JSON:
      ```python
      prophet_ready = is_prophet_warmed_up()
      ready = models_loaded and prophet_ready

      if not ready:
          response.status_code = 503

      return {
          "status": "healthy" if ready else "starting",
          "models_loaded": models_loaded,
          "prophet_warmed_up": prophet_ready,
          "version": "1.0.0",
      }
      ```

       ```python
       # Replace:
       result = no_show_model.predict(features)

       loop = asyncio.get_event_loop()
       result = await loop.run_in_executor(_thread_pool, no_show_model.predict, features)
       ```

    4. In `predict_clv()` — same pattern, wrap `clv_model.predict(features)`:
       ```python

       ```


       ```python

       ```

    3. In `get_dynamic_pricing()` — wrap `pricing_model.get_optimal_price(...)` call in executor:
       ```python
       loop = asyncio.get_event_loop()
       result = await loop.run_in_executor(
           _thread_pool,
           lambda: pricing_model.get_optimal_price(
               service_id=request.service_id,
               price_min=request.price_min,
               price_max=request.price_max,
               hour_of_day=request.hour_of_day,
               day_of_week=request.day_of_week,
               utilization=request.utilization,
               base_price=request.base_price,
           )
       )
       ```
       Note: Use lambda because `get_optimal_price` has keyword arguments; `run_in_executor` only passes positional args.
