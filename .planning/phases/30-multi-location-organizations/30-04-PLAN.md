**Create `apps/web/components/layout/location-switcher.tsx`:**

A dropdown component that:
1. On mount, calls `GET /api/v1/organizations` to check if user belongs to an org
2. If no org, renders nothing (null)
3. If org exists, renders a shadcn/ui `DropdownMenu` (or `Select`) showing:
   - Current location name (from auth store user.companyName)
   - Divider
   - List of all locations in the org (name + city)
   - Active location has a check icon
   - Deactivated locations shown grayed out (not selectable)
4. When user selects a different location:
   - Calls `POST /api/v1/auth/switch-location` with `{ company_uuid }`
   - On success: updates auth store with new accessToken, updates user.companyId and user.companyName
   - Calls `window.location.reload()` to refresh all data with new company context (simplest approach — TanStack Query cache is company-scoped so full reload is cleaner than selective invalidation)
   - On error: show toast notification via sonner

Use shadcn/ui components: `DropdownMenu`, `DropdownMenuTrigger`, `DropdownMenuContent`, `DropdownMenuItem`, `DropdownMenuSeparator`. Import `Building2` and `ChevronDown` icons from lucide-react.

Style: match existing header component patterns. Button trigger with company name + chevron, max-width 200px, text truncation for long names.

Use React Query (`useQuery`) for fetching org data with key `['organization']` and `staleTime: 5 * 60 * 1000` (5 min — org data rarely changes).

**Update `apps/web/stores/auth.store.ts`:**

Add to the `User` interface:
```typescript
companyName: string; // Already exists but ensure it's populated
organizationId?: string; // Organization UUID if user belongs to org
```

Add `switchLocation` action:
```typescript
switchLocation: async (companyUuid: string) => {
  const response = await apiClient.post<SwitchLocationResponse>('/auth/switch-location', {
    company_uuid: companyUuid,
  });
  set({
    accessToken: response.access_token,
    user: {
      ...get().user!,
      companyId: response.company.uuid,
      companyName: response.company.name,
    },
  });
};
```
