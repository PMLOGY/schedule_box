Create `packages/database/src/schema/organizations.ts` with two tables following existing Drizzle patterns (see auth.ts for reference):

**organizations table:**
- `id`: serial PK
- `uuid`: uuid, defaultRandom, notNull, unique
- `name`: varchar(255), notNull
- `slug`: varchar(255), notNull, unique
- `ownerUserId`: integer FK to users.id (onDelete: 'restrict') — the user who created the org
- `maxLocations`: integer, default(1) — enforced at API level, not DB constraint
- `isActive`: boolean, default(true)
- `createdAt`: timestamp with timezone, defaultNow, notNull
- `updatedAt`: timestamp with timezone, defaultNow, notNull
- Indexes: `idx_organizations_slug` on slug, `idx_organizations_owner` on ownerUserId

**organization_members table:**
- `id`: serial PK
- `organizationId`: integer FK to organizations.id (onDelete: 'cascade'), notNull
- `userId`: integer FK to users.id (onDelete: 'cascade'), notNull
- `companyId`: integer FK to companies.id (onDelete: 'cascade'), nullable — null means access to ALL locations (franchise_owner). Non-null means scoped to specific location (location_manager).
- `role`: varchar(30), notNull, with CHECK constraint: `role IN ('franchise_owner', 'location_manager')`
- `createdAt`: timestamp with timezone, defaultNow, notNull
- Unique constraint: `org_members_org_user_company_unique` on (organizationId, userId, companyId)
- Indexes: `idx_org_members_org` on organizationId, `idx_org_members_user` on userId, `idx_org_members_company` on companyId

**Modify companies table in auth.ts:**
Add `organizationId` column:
- `organizationId`: integer FK to organizations.id (onDelete: 'set null'), nullable, default null
- Add index: `idx_companies_organization` on organizationId

**Create `packages/shared/src/types/organization.ts`:**

```typescript
/**
 * Multi-Location Organization Types
 * Used by organization management, location switching, and org dashboards
 */

export type OrgRole = 'franchise_owner' | 'location_manager';

export interface Organization {
  id: number;
  uuid: string;
  name: string;
  slug: string;
  owner_user_id: number;
  max_locations: number;
  is_active: boolean;
  created_at: string;
  updated_at: string;
}

export interface OrganizationMember {
  id: number;
  organization_id: number;
  user_id: number;
  company_id: number | null; // null = all locations (franchise_owner)
  role: OrgRole;
  created_at: string;
}

/** API response shape for organization with locations */
export interface OrganizationWithLocations extends Organization {
  locations: OrganizationLocation[];
  member_count: number;
}

/** Location summary within an organization */
export interface OrganizationLocation {
  company_uuid: string;
  company_name: string;
  company_slug: string;
  address_city: string | null;
  is_active: boolean;
}

/** Location metrics for org dashboard */
export interface LocationMetrics {
  company_uuid: string;
  company_name: string;
  bookings_count: number;
  revenue_total: string;
  occupancy_percent: number | null;
}

/** Switch location request */
export interface SwitchLocationRequest {
  company_uuid: string;
}

/** Switch location response */
export interface SwitchLocationResponse {
  access_token: string;
  refresh_token: string;
  expires_in: number;
  company: {
    uuid: string;
    name: string;
    slug: string;
  };
}
```
