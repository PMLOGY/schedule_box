    Create `apps/web/app/api/v1/gift-cards/[id]/balance/route.ts`:

    **GET /api/v1/gift-cards/[id]/balance**:
    - Use createRouteHandler with paramsSchema: giftCardIdParamSchema, requiredPermissions: [PERMISSIONS.COUPONS_MANAGE]
    - Call findCompanyId(user.sub)
    - Find gift card by UUID and companyId. If not found, throw NotFoundError
    - Return successResponse with: { id: giftCard.uuid, code: giftCard.code, initial_balance: Number(giftCard.initialBalance), current_balance: Number(giftCard.currentBalance), currency: giftCard.currency, is_active: giftCard.isActive, valid_until: giftCard.validUntil }

    Create `apps/web/app/api/v1/gift-cards/redeem/route.ts`:

    **POST /api/v1/gift-cards/redeem**:
    - Use createRouteHandler with bodySchema: giftCardRedeemSchema, requiredPermissions: [PERMISSIONS.COUPONS_MANAGE]
    - Call findCompanyId(user.sub)
    - Execute within db.transaction():
      1. SELECT FROM giftCards WHERE code = body.code (already uppercase from Zod) AND companyId AND isActive = true, using `.for('update')` to lock the row
      2. If not found, throw NotFoundError('Gift card not found')
      3. Check expiration: if validUntil exists and new Date() > validUntil, throw ValidationError('Gift card has expired')
      4. Check balance: if Number(currentBalance) < body.amount, throw ValidationError('Insufficient gift card balance')
      5. Calculate newBalance = Number(giftCard.currentBalance) - body.amount
      6. UPDATE giftCards SET currentBalance = String(newBalance), updatedAt = new Date() WHERE id = giftCard.id
      7. INSERT into giftCardTransactions: giftCardId: giftCard.id, bookingId: body.booking_id (or null), type: 'redemption', amount: String(body.amount), balanceAfter: String(newBalance)
    - Return successResponse with: { id: giftCard.uuid, current_balance: newBalance, amount_redeemed: body.amount, currency: giftCard.currency }

    CRITICAL: The SELECT FOR UPDATE is essential to prevent race conditions where two concurrent redemptions could overdraw the balance. The transaction ensures the balance check and deduction happen atomically.

    Import ValidationError and NotFoundError from @schedulebox/shared. Import giftCards and giftCardTransactions from @schedulebox/database.
