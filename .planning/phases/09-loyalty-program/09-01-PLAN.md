  <action>
Create `packages/shared/src/schemas/loyalty.ts` with these Zod schemas:

1. **loyaltyProgramCreateSchema** — Per API spec (lines 3556-3570):
   - `name`: z.string().min(1).max(255) (required)
   - `description`: z.string().max(2000).optional()
   - `type`: z.enum(['points', 'stamps', 'tiers']) (required)
   - `points_per_currency`: z.number().positive().max(99999).default(1) (optional)

2. **loyaltyProgramUpdateSchema** — Partial of create:
   - `name`: z.string().min(1).max(255).optional()
   - `description`: z.string().max(2000).optional()
   - `type`: z.enum(['points', 'stamps', 'tiers']).optional()
   - `points_per_currency`: z.number().positive().max(99999).optional()
   - `is_active`: z.boolean().optional()

3. **loyaltyCardCreateSchema** — Per API spec (lines 3596-3610):
   - `customer_id`: z.string().uuid() (required, customer UUID)

4. **addPointsSchema** — Per API spec (lines 3637-3658):
   - `points`: z.number().int().positive() (required)
   - `description`: z.string().max(255).optional()

5. **redeemRewardSchema**:
   - `card_id`: z.string().uuid() (required, card UUID)

6. **rewardCreateSchema**:
   - `name`: z.string().min(1).max(255) (required)
   - `description`: z.string().max(2000).optional()
   - `points_cost`: z.number().int().positive() (required)
   - `reward_type`: z.enum(['discount_percentage', 'discount_fixed', 'free_service', 'gift']) (required)
   - `reward_value`: z.number().positive().optional() (required for discount types)
   - `applicable_service_id`: z.string().uuid().optional()
   - `max_redemptions`: z.number().int().positive().optional()

7. **rewardUpdateSchema** — Partial of create fields plus `is_active`

8. **loyaltyCardListQuerySchema**:
   - `page`: z.coerce.number().int().min(1).default(1)
   - `limit`: z.coerce.number().int().min(1).max(100).default(20)
   - `customer_id`: z.string().uuid().optional()

9. **transactionListQuerySchema**:
   - `page`: z.coerce.number().int().min(1).default(1)
   - `limit`: z.coerce.number().int().min(1).max(100).default(20)
   - `type`: z.enum(['earn', 'redeem', 'expire', 'adjust', 'stamp']).optional()

10. **tierCreateSchema**:
    - `name`: z.string().min(1).max(100) (required)
    - `min_points`: z.number().int().min(0) (required)
    - `benefits`: z.record(z.unknown()).optional()
    - `color`: z.string().regex(/^#[0-9A-Fa-f]{6}$/).default('#3B82F6')
    - `sort_order`: z.number().int().min(0).default(0)

11. **tierUpdateSchema** — Partial of tierCreateSchema

Export each schema and its inferred type using `z.infer`. Use camelCase with `Schema` suffix for names.

Update `packages/shared/src/schemas/index.ts` to re-export all schemas from loyalty.ts. Follow existing pattern from booking.ts exports.
