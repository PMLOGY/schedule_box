  <files>

  <action>
Create `packages/events/src/events/loyalty.ts` following the exact pattern of `packages/events/src/events/booking.ts`:

Constants:
- `EVENT_SOURCE = 'loyalty-service'`
- `EVENT_TYPE_PREFIX = 'com.schedulebox.loyalty'`

Event payloads:

1. **LoyaltyCardCreatedPayload**:
   - `cardUuid: string` — Loyalty card UUID
   - `companyId: number` — Tenant ID
   - `customerUuid: string` — Customer UUID
   - `programUuid: string` — Program UUID
   - `cardNumber: string` — Card number

2. **PointsEarnedPayload**:
   - `cardUuid: string` — Loyalty card UUID
   - `companyId: number` — Tenant ID
   - `customerUuid: string` — Customer UUID
   - `points: number` — Points earned (positive)
   - `balanceAfter: number` — New balance after earning
   - `bookingUuid: string | null` — Booking that triggered earning (null for manual)
   - `description: string` — Reason for points

3. **TierUpgradedPayload**:
   - `cardUuid: string` — Loyalty card UUID
   - `companyId: number` — Tenant ID
   - `customerUuid: string` — Customer UUID
   - `previousTierName: string | null` — Previous tier (null if no tier)
   - `newTierName: string` — New tier name
   - `newTierMinPoints: number` — Min points for new tier

4. **RewardRedeemedPayload**:
   - `cardUuid: string` — Loyalty card UUID
   - `companyId: number` — Tenant ID
   - `customerUuid: string` — Customer UUID
   - `rewardId: number` — Reward ID
   - `rewardName: string` — Reward name
   - `pointsSpent: number` — Points deducted
   - `balanceAfter: number` — New balance after redemption

Type aliases (same pattern as booking events):
- `LoyaltyCardCreatedEvent = CloudEvent<LoyaltyCardCreatedPayload>`
- `PointsEarnedEvent = CloudEvent<PointsEarnedPayload>`
- `TierUpgradedEvent = CloudEvent<TierUpgradedPayload>`
- `RewardRedeemedEvent = CloudEvent<RewardRedeemedPayload>`

Factory functions (same pattern as booking events):
- `createLoyaltyCardCreatedEvent(data) -> LoyaltyCardCreatedEvent`
- `createPointsEarnedEvent(data) -> PointsEarnedEvent`
- `createTierUpgradedEvent(data) -> TierUpgradedEvent`
- `createRewardRedeemedEvent(data) -> RewardRedeemedEvent`

Each factory uses `createCloudEvent(EVENT_TYPE_PREFIX + '.card_created', EVENT_SOURCE, data, data.cardUuid)` pattern.

Routing keys will be: `loyalty.card_created`, `loyalty.points_earned`, `loyalty.tier_upgraded`, `loyalty.reward_redeemed`.
