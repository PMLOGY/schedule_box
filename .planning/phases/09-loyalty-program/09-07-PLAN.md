**TanStack Query Hooks** (`apps/web/hooks/use-loyalty-queries.ts`):

Create hooks following the pattern from `apps/web/hooks/use-bookings-query.ts`:

Query hooks (read operations):
1. `useLoyaltyProgram()` — GET /api/v1/loyalty/programs, returns company's loyalty program with tiers
2. `useLoyaltyCards(params?: { page?, limit?, customer_id? })` — GET /api/v1/loyalty/cards with pagination
3. `useLoyaltyCard(id: string)` — GET /api/v1/loyalty/cards/:id with tier progress
4. `useRewards(params?: { page?, limit?, is_active? })` — GET /api/v1/loyalty/rewards with pagination
5. `useTransactions(cardId: string, params?: { page?, limit?, type? })` — GET /api/v1/loyalty/cards/:id/transactions
6. `useTiers()` — GET /api/v1/loyalty/tiers

Mutation hooks (write operations):
7. `useCreateProgram()` — POST /api/v1/loyalty/programs, invalidates program query
8. `useUpdateProgram()` — PUT /api/v1/loyalty/programs, invalidates program query
9. `useCreateReward()` — POST /api/v1/loyalty/rewards, invalidates rewards query
10. `useUpdateReward()` — PUT /api/v1/loyalty/rewards/:id, invalidates rewards query
11. `useAddPoints()` — POST /api/v1/loyalty/cards/:id/add-points, invalidates card + transactions queries
12. `useRedeemReward()` — POST /api/v1/loyalty/rewards/:id/redeem, invalidates card + rewards queries
13. `useCreateCard()` — POST /api/v1/loyalty/cards, invalidates cards query
14. `useCreateTier()` — POST /api/v1/loyalty/tiers, invalidates program + tiers queries

Use `@tanstack/react-query` with proper query keys like `['loyalty', 'program']`, `['loyalty', 'cards', params]`, etc. Each mutation should invalidate relevant queries on success.

**Zustand Store** (`apps/web/stores/loyalty.store.ts`):

Create a minimal Zustand store for UI state:
- `selectedCardId: string | null` — Currently selected card in list
- `programFormOpen: boolean` — Whether program create/edit form is open
- `rewardFormOpen: boolean` — Whether reward create/edit form is open
- `addPointsDialogOpen: boolean` — Whether add-points dialog is open
- `editingRewardId: string | null` — Reward being edited (null for create)
- Actions: `setSelectedCardId`, `openProgramForm`, `closeProgramForm`, `openRewardForm`, `closeRewardForm`, `openAddPointsDialog`, `closeAddPointsDialog`, `setEditingRewardId`, `reset`

Follow the same pattern as `apps/web/stores/booking-wizard.store.ts`.
