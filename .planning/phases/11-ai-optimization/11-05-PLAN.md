    **apps/web/hooks/useOptimization.ts:** Create TanStack Query hooks for all 4 optimization endpoints. Follow the exact same pattern used in existing hooks (check apps/web/hooks/ for reference patterns like useLoyaltyProgram, useTransactions, etc.):

    ```typescript
    'use client';
    import { useQuery, useMutation } from '@tanstack/react-query';
    import { apiClient } from '@/lib/api-client';
    ```

    `useUpselling(serviceId: number | null, customerHistory?: number[])`:
    - queryKey: `['ai', 'optimization', 'upselling', serviceId]`
    - queryFn: POST to `/ai/optimization/upselling` with `{ customer_id: 0, current_service_id: serviceId, customer_history: customerHistory }`. Note: customer_id=0 is used for anonymous/new customers; the API handles this gracefully.
    - enabled: `!!serviceId` (only fetch when a service is selected)
    - staleTime: 60_000 (1 minute — recommendations don't change rapidly)
    - retry: false (use fallback, don't retry AI calls)
    - Return query result.

    `useDynamicPricing(serviceId: number, priceMin: number, priceMax: number, options?: { hourOfDay?: number, dayOfWeek?: number, utilization?: number, basePrice?: number })`:
    - queryKey: `['ai', 'optimization', 'pricing', serviceId, options?.hourOfDay, options?.dayOfWeek]`
    - queryFn: POST to `/ai/optimization/pricing` with all fields.
    - Defaults: hourOfDay = new Date().getHours(), dayOfWeek = new Date().getDay(), utilization = 0.5.
    - staleTime: 300_000 (5 minutes — pricing changes slowly)
    - Return query result.

    `useCapacityForecast(companyId: number, daysAhead: number = 7, currentCapacity: number = 8)`:
    - queryKey: `['ai', 'optimization', 'capacity', companyId, daysAhead]`
    - queryFn: POST to `/ai/optimization/capacity` with all fields.
    - staleTime: 600_000 (10 minutes — forecast is relatively stable)
    - Return query result.

    `useReminderTiming(customerId: number, channel: 'email' | 'sms' | 'push' = 'email')`:
    - queryKey: `['ai', 'optimization', 'reminder-timing', customerId, channel]`
    - queryFn: POST to `/ai/optimization/reminder-timing` with fields.
    - enabled: `!!customerId`
    - staleTime: 300_000 (5 minutes)
    - Return query result.
