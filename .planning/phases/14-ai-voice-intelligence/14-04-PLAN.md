1. **Create services/ai/app/services/scraper.py** -- Competitor data extraction:

   ```python
   import logging
   import asyncio
   from typing import Optional
   import httpx
   from bs4 import BeautifulSoup
   from ..config import settings

   logger = logging.getLogger(__name__)

   SCRAPE_TIMEOUT = 15.0  # 15s per request
   SCRAPE_DELAY = 5.0  # 5s between requests to same domain (polite scraping)
   USER_AGENT = "Mozilla/5.0 (compatible; ScheduleBox/1.0; +https://schedulebox.cz)"
   ```

   Class `CompetitorScraper`:
   - Constructor takes optional `httpx.AsyncClient` (for testing/reuse)
   - Uses httpx (already in requirements) NOT aiohttp (per research recommendation to minimize deps)

   Methods:

   **`async def scrape_website(self, url: str) -> dict`:**
   - GET the URL with follow_redirects=True, timeout=SCRAPE_TIMEOUT, headers with User-Agent
   - Parse HTML with BeautifulSoup
   - Extract pricing data: look for common patterns in Czech booking sites
     - Tables with price-like content (numbers + CZK/Kc)
     - Service name + price pairs from lists
     - Structured data (JSON-LD, microdata) if available
   - Extract service names: look for service lists, menu items, catalog entries
   - Return `{"services": [...], "pricing": [...], "source_url": url, "raw_title": soup.title.string}`
   - On error (timeout, connection, parsing): return `{"error": str(e), "services": [], "pricing": []}`
   - Respect polite scraping: asyncio.sleep(SCRAPE_DELAY) after each request

   **`async def scrape_google_reviews(self, business_name: str, location: str) -> dict`:**
   - Uses Google Places API (NOT direct scraping) for ToS compliance
   - If GOOGLE_PLACES_API_KEY is empty, return `{"error": "google_places_api_key_not_configured", "reviews": {}}`
   - Call Places API Find Place to get place_id
   - Call Places API Details to get rating, review_count, total_ratings
   - Return ONLY aggregate data: `{"average_rating": 4.3, "total_reviews": 127, "business_name": "..."}`
   - NEVER store individual review text or reviewer names (GDPR)
   - On error: return `{"error": str(e), "reviews": {}}`

   **`async def scrape_competitor(self, name: str, url: str, data_types: list[str]) -> list[dict]`:**
   - Orchestrator method: for each data_type, call appropriate scraper
   - "pricing" and "services" -> scrape_website(url)
   - "reviews" -> scrape_google_reviews(name, location from URL domain)
   - Returns list of `{"competitor_name": name, "data_type": type, "data": {...}, "scraped_at": iso_now}`
   - Rate limit: max 3 data types per call

2. **Create services/ai/app/routers/competitor.py** -- FastAPI router:

   **POST /competitor/scrape** -- Trigger scraping for a competitor:
   ```python
   @router.post("/scrape", response_model=CompetitorScrapeResponse)
   async def scrape_competitor(request: CompetitorScrapeRequest):
       """
       Trigger scraping of a competitor's public website data.

       Extracts pricing, services, and/or review data from public sources.
       Uses Google Places API for reviews (not direct scraping).
       GDPR-safe: only collects aggregate business data.

       Note: This is a synchronous scraping operation (takes 5-30s).
       For production, consider moving to a background job.
       """
       try:
           scraper = CompetitorScraper()
           results = await scraper.scrape_competitor(
               name=request.competitor_name,
               url=request.competitor_url,
               data_types=request.data_types,
           )
           return CompetitorScrapeResponse(
               results=[CompetitorScrapeResult(**r) for r in results],
               errors=[],
               fallback=False,
           )
       except Exception as e:
           logger.error(f"Competitor scraping failed: {e}")
           return CompetitorScrapeResponse(
               results=[],
               errors=[str(e)],
               fallback=True,
           )
   ```

   ```typescript
   // COMPETITOR_MONITORS TABLE (Phase 14)

   export const competitorMonitors = pgTable(
     'competitor_monitors',
     {
       id: serial('id').primaryKey(),
       companyId: integer('company_id')
         .notNull()
         .references(() => companies.id, { onDelete: 'cascade' }),
       competitorName: varchar('competitor_name', { length: 255 }).notNull(),
       competitorUrl: varchar('competitor_url', { length: 500 }).notNull(),
       scrapeFrequency: varchar('scrape_frequency', { length: 20 }).notNull().default('weekly'),
       isActive: boolean('is_active').notNull().default(true),
       lastScrapedAt: timestamp('last_scraped_at', { withTimezone: true }),
       createdAt: timestamp('created_at', { withTimezone: true }).defaultNow(),
       updatedAt: timestamp('updated_at', { withTimezone: true }).defaultNow(),
     },
     (table) => ({
       frequencyCheck: check(
         'competitor_monitors_frequency_check',
         sql`${table.scrapeFrequency} IN ('daily', 'weekly', 'monthly')`,
       ),
       companyIdx: index('idx_competitor_monitors_company').on(table.companyId),
     }),
   );
   ```

     ```typescript
     handler: async ({ body, companyId }) => {
       try {
         const request = { ...body, company_id: companyId };
         const result = await triggerCompetitorScrape.fire(request);

         // If scraping succeeded, store results in competitor_data table
         if (result.results.length > 0 && companyId) {
           // Insert scraped results into competitor_data using Drizzle
           // Import db and competitorData from @schedulebox/database
           for (const item of result.results) {
             await db.insert(competitorData).values({
               companyId: companyId,
               competitorName: item.competitor_name,
               competitorUrl: body.competitor_url,
               dataType: item.data_type,
               data: item.data,
             });
           }
         }

         return successResponse(result);
       } catch {
         const fallback = getCompetitorScrapeFallback({
           ...body,
           company_id: companyId ?? 0,
         });
         return successResponse(fallback);
       }
     }
     ```
